# verificando-ip-alvo
Este é um script em Python focado em automação de rede e reconhecimento (Recon).  A ferramenta recebe a URL de um site alvo, limpa o endereço automaticamente tirando o 'https' e as barras extras, descobre o IP real do servidor e realiza um teste de Ping para validar se o alvo está ON ou OFF.  No final, os resultados são salvos em uma log.txt
